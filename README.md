
# mobile_dev_com_dashboard

Mobile dev dashboard project challenge.

## Getting Started

This project is a development challenge with `MobileDev.Com` community.
A challenge on charts in Flutter.
4 types of charts were considered.
1. BarChart
2. PieChart
3. LineChart
4. Image-chart
